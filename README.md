

# 基于多层感知机(MLP)的时间序列预测


### 数据集

sp500.csv文件包含2000年1月至2018年9月标准普尔500股票指数的开盘价，最高价，最低价，收盘价以及每日交易量。我们针对收盘价(CLOSE)进行预测。

数据集被划分为不相交的训练集和测试集，比例为8：2，训练集包含3758条数据，测试集包含939条数据。
